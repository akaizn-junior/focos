Focos

Focos tracks elements on the screen in a grid of a defined size. You can navigate through the grid by using different keys. Focos uses arrow keys for navigation by default, but different bindings may be provided. Multiple bindings to the same action are allowed. Navigate through the grid below using the bindings defined on the code snippet. 
                window.focos({
                  gridSize: "5x4",
                  step: 1,
                  initialFocus: [0, 0],
                  focusOnClick: true,
                  keys: {
                    s: "up",
                    z: "down",
                    a: "left",
                    d: "right"
                  }
                });
