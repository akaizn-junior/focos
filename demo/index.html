Focos

Focos is a navigation library that tracks elements on the screen defined in a grid of a certain size. You can navigate through the grid by using different keys. For navigation Focos uses arrow keys by default, but different bindings may be provided. Multiple bindings to the same action are allowed. You may navigate through the grid below using the bindings defined on the code snippet.

How it works

Focos uses css classes, the .focus method on DOM elements and tabindex to focus items. Focos sets the following css classes: focusable, focos-cell(number) and focused. By using a combination of approaches, different scenarios may be covered when using the library. Scenario number 1, the user may want to add focus to menu items on a menu, but still use links

Usage

Get it via jsDelivr or download it from the repository. Links below. Load the library and call it as follows.

<script src="https://cdn.jsdelivr.net/gh/verdebydesign/focos/dist/latest/focos.min.js"></script>
<script>
  window.focos({/* Settings */});
<script>

Settings

Focos requires a settings object, for initial setup. Let's look at the available options: gridSize The only required setting for the library. Focos will use the grid size value to construct a grid for navigation through the elements located on specific cells. initialFocus, this cell gets initial focus, as soon as the page loads. step the distance between a point to the next. focusOnClick when this flag is on, cells are focusable on mouse down. keys key bindings for navigation. Multiple keys may be bound to an orientation. A key name such as 'a' or 'ArrowUp' or key code may be provided as a binding. 

                window.focos({
                gridSize: "5x4",
                step: 1,
                initialFocus: [0, 4],
                focusOnClick: true,
                keys: {
                    s: "up",
                    z: "down",
                    a: "left",
                    d: "right",
                    37: "up",
                    38: "down",
                    39: "left",
                    40: "right"
                 }
                });
